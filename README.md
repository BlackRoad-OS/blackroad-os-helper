## 🛠 Developer Quick Start

If you want to run or extend the Worker locally:

```bash
# 1. Clone
git clone https://github.com/BlackRoad-OS/blackroad-os-helper.git
cd blackroad-os-helper

# 2. Install
npm install

# 3. Run locally
npm run dev
```

The Worker opens at `http://localhost:8787` — the same visual portal you see live.

**Deploy to Cloudflare:**
```bash
npm run deploy
```
